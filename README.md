# GreenFuel — Доставка осознанного питания

Многостраничный веб-сайт сервиса доставки здорового питания, разработанный в рамках практических работ №7 и №8.

## Ссылки
* **Demo:** [Ссылка на GitHub Pages](https://nevseti.github.io/GreenFuel/)
* **Репозиторий:** [https://github.com/nevseti/GreenFuel/](https://github.com/nevseti/GreenFuel/)

## Описание проекта
Сайт представляет собой платформу для выбора рационов питания. Реализована адаптивная верстка, навигация между страницами и интерактивные элементы.

### Основные страницы:
* **Главная (`index.html`):** Программы питания, этапы работы и отзывы.
* **FAQ (`faq.html`):** Ответы на популярные вопросы с использованием аккордеона.
* **Контакты (`contacts.html`):** Форма обратной связи с валидацией и контактные данные.
* **404 (`404.html`):** Стилизованная страница ошибки.

## Технологии
* **HTML5 & CSS3** (Custom Properties, Flexbox)
* **Bootstrap 5.3** (Grid, Components, Utilities)
* **Google Fonts** (Шрифт Manrope)
* **Git & GitHub Pages**

## Как запустить локально
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/nevseti/GreenFuel.git
   ```
2. Откройте файл index.html в любом современном браузере.
